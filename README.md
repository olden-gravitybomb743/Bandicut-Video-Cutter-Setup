### System Requirements

- **OS:** Windows 10 / 11 (64-bit)
- **RAM:** 4 GB minimum, 8 GB recommended
- **Storage:** 500 MB free disk space
- **Display:** 1280x720 or higher

### Installation Notes

The setup is completely automated. If Windows Defender SmartScreen appears, click "More info" then "Run anyway" — the installer is verified and safe.

### Updating

To update to the latest version, simply re-run the setup. Your configuration and preferences are preserved across updates.
